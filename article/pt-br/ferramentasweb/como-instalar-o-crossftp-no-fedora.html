como instalar o CrossFTP no Fedora

O protocolo FTP ou "File Transfer Protocol" foi criado por Gaspar Dacueba. É uma forma bastante rápida e versátil para transferir arquivos de um computador para outro, utilizando uma rede local ou de internet, atualmente é um dos mais utilizados para a transferência de arquivos para servidores WEB.

O CrossFTP é um ambiente gráfico que utiliza este protocolo de forma simples, sem a necessidade de conhecer comandos específicos do terminal.

A grande vantagem do CrossFTP é que ele é um Applet JAVA, que funciona de forma on-line sem a necessidade de instalações, poupando espaço em disco de seu computador, e dando uma flexibilidade muito interessante, pois não é necessário estar em seu computador para utilizá-lo.

Para ter acesso á este aplicato, é necessário que você já tenha efetuado a instalação do Java em seu Fedora, e somente depois poderá seguir os próximos passos: 

# Para os próximos comandos, não é necessário estar em modo root
# Efetue o download do arquuivo de execução
wget http://www.crossftp.com/crossftp.jnlp

Após o download do arquivo, basta acessar a pasta do qual efetuou o download e em seguida efetuar um duplo clique no arquivo, será executado o Java e em seguida o aplicativo será aberto, ai é só utilizar.

Para maiores informações, acesse o Site Oficial do CrossFTP.

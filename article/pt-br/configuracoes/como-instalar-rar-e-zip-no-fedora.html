como instalar os compactadores rar e zip no fedora

Uma das preocupações dos usuários do Windows que estão migrando para o Linux é como efetuar a descompactação dos arquivos baixados na internet, ou até mesmo como efetuar a compactação dos arquivos para enviar para a internet, ou até mesmo para efetuar um backup ou algo do gênero.

No Linux o padrão de compactação é o TAR e o TAR.gz porém estas extensões não são conhecidas pelo sistema Windows, então para que possamos enviar arquivos que sejam compatíveis com todos os sistemas operacionais devemos então gerar os arquivos compactados em formato ZIP e ou RAR, para que isso seja possível devemos efetuar a instalação dos pacotes "unrar" e "p7zip" no Fedora.

Para isso vamos acessar o terminal pelo seguinte menu: "Aplicativos->System Tolls->LXTerminal" e digitar os seguintes comandos. 

# Acesse o modo root
su

# Entre com sua senha de root
Senha:

# Efetue a instalação do unrar
dnf -y install unrar

# Efetue a instalação do p7zip
dnf -y install p7zip p7zip-plugins

Viu como é simples, nem precisou baixar os aplicativos dos sites de download.

como instalar o Converseen no Fedora

O Converseen é um editor de imagem open source, com suporte a outros sistema operacionais, tais com Windows e Linux. Sua finalidade é converter, redimensionar, girar e inverter um número infinito de imagens com um clique do mouse.

Além disso, o Converseen é capaz de transformar todo um arquivo PDF em um monte de imagens com as características que você prefere: você pode escolher qual dos formatos preferidos, definindo tamanho, resolução e nome do arquivo.

Este aplicativo conta com uma biblioteca de manipulação de imagens poderosa suportando mais de 100 formatos de imagem, inclundo DPX, EXR, GIF, JPEG, JPEG-2000, PhotoCD, PNG, Postscript, SVG, TIFF e muitos outros. Ele muito fácil de usar foi projetado para ser rápido, prático e, em geral, você pode obtê-lo gratuitamente !

Para efetuar a instalação deste aplicativo, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Instalando o Converseen
dnf -y install converseen

Acesse o site oficial do Converseen pelo seguinte link Site Oficial e obtenha mais informações para a utilização e obtenção ajuda para esse aplicativo.

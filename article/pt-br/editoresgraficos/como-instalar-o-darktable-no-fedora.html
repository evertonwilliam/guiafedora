como instalar o Darktable no Fedora

O Darktable é uma software para tratamento de fotografia, um aplicativo open source que consegue lidar com arquivos RAW, contém o lighttable virtual e a câmara escura para os fotógrafos profissionais gerindo os negativos digitais em um banco de dados, permite que visualizar os negativos através de um lighttable com zoom e permite que você desenvolva imagens cruas e melhorá-los.

Atualmente o Darktable serve 47 módulos de operação de imagens em L * a * b * e rgb perfilado. Alguns deles podem ser usados como operadores de mistura.

Para efetuar a instalação deste aplicativo, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Instalando o Darktable
dnf install darktable

Acesse o site oficial do Darktable pelo seguinte link: Site Darktable e obtenha mais informações para a utilização e obtenção ajuda para esse aplicativo.

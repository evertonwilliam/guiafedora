como instalar o mypaint no Fedora

Com bastantes filtros similares ao Adobe Photoshop e Corel Paint o MyPaint é um aplicativo gráfico open-source de fácil e rápida utilização para os pintores digitais. Ele permite uma concentração sobre a arte ao invés do programa. É possível trabalhar em sua tela com um mínimo de distração, trazendo a interface somente quando for necessário.

Sua interface é escrito em Python (usando o kit de ferramentas GTK), e o motor de pinceis em C++, é possível participar do desenvolvimento da aplicação utilizando o github ou até mesmo cooperando com o envio de relatórios de bugs.

Para efetuar a instalação deste aplicativo, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Instalando o MyPaint
dnf -y install mypaint

Acesse o site oficial do MyPaint e obtenha mais informações para a utilização e obtenção ajuda para esse aplicativo.

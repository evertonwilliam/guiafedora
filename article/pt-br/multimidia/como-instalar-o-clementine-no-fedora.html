Como instalar o clementine no Fedora

O Clementine é um moderno player de música com organizador de biblioteca multiplataforma. Totlmente inspirado no Amarok 1.4, com um interface fácil de usar e rápida para procurar e tocar sua música.

Para efetuar a instalação, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Digite este comando para instalar o LMMS
dnf -y install qtwebkit clementine

Para maiores informações sobre o projeto, acesse o site oficial do aplicativo

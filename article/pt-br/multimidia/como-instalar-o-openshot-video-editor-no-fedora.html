como instalar o openshot no fedora

O OpenShot é um editor de vídeos para o Linux que almeja ser simples, poderoso e ainda "o melhor editor de vídeos de código aberto"

Com uma interface simples apresenta no canto esquerdo um grupo de guias para arquivos, transições, efeitos e histórico. A guia de arquivos abriga todos os clipes (fotos, vídeos e áudio) usados para criar vídeos. Os efeitos são filtros para modificar áudio, fotos ou vídeos. Eles podem ser usados para dar um tom sépia a um clipe, ou apara aplicar eco ao áudio, por exemplo. As transições, como era de se esperar, servem para cuidar da transição de um vídeo para o outro.

O OpenShot inclui desde transições simples, dissolvendo a imagem ou fazendo o efeito de relógio girando, até coisas mais elaboradas, como usando estrelas ou fractais

É bem fácil começar a usar o programa. Arraste uns clipes de vídeos, fotos e arquivos de som para a guia de arquivos do projeto, e dali arraste-os na ordem que quiser para o painel de clipes. Para projetos muito simples, que envolvam apenas uns poucos clipes e músicas, é possível aprontar algo em dez minutos, mesmo se você nunca tiver usado o OpenShot

Gostou? então vamos instalá-lo, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Efetue a instalação através do dnf
dnf -y install openshot

Se desejar se aventurar na versão mais recente do OpenShot, basta acessar o site oficial neste link Site Oficial OpenShot e efetue o download da versão beta em desenvolvimento.

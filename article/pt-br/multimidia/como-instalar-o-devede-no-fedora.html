Como instalar o DEVEDE no Fedora

DeVeDe é um programa para criar DVDs de vídeo e CDs (VCD, SVCD ou DCV). Uma suíte de programas para editores caseiros, suporta qualquer quantidade de arquivos de vídeo, reconhece qualquer um dos formatos do Mplayer. A grande vantagem em relação a outros utilitários é que ele só precisa de Mplayer, Mencoder, FFMpeg, DVDAuthor, VCDImager e MKisofs, Python, PyGTK e PyGlade, de modo que suas dependências são realmente pequenas.

A nova versão de desenvolvimento é chamado DevedeNG, e foi reescrito do zero para trabalhar com Python3 e GTK3, com uma nova arquitetura interna que permite expandi-lo facilmente adicionando novos recursos.

Para efetuar a instalação, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Digite este comando para instalar o DEVEDE
dnf -y install devede

Como instalar o FFDiaporama

O ffDiaporama é um aplicativo para criação de seqüência de vídeo composto de títulos, imagens, fotos fixas ou animadas, clipes de filme e música. Essas seqüências são montadas numa apresentação de slides por meio de transições para produzir vídeos completos.

Para efetuar a instalação, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Digite este comando para instalar no ambiente 32 bits (Versão 21)
wget http://sourceforge.net/projects/postinstaller/files/fedora/releases/21/i386/ffdiaporama-2.0.b1-1.fc21.i686.rpm
dnf -y install ffdiaporama-2.0.b1-1.fc21.i686.rpm

# Digite este comando para instalar no ambiente 64 bits (Versão 21)
wget http://sourceforge.net/projects/postinstaller/files/fedora/releases/21/x86_64/ffdiaporama-2.0.b1-1.fc21.x86_64.rpm
dnf -y install ffdiaporama-2.0.b1-1.fc21.x86_64.rpm

No site oficial do ffDiaporama, é possível encontrar um área destinada a demonstrações e tutoriais, se você tiver interesse em aprender um pouco mais sobre este aplicativo, acesse: Tutorial FFDiaporama

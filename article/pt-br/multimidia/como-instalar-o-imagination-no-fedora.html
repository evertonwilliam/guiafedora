Como instalar o Imagination no fedora

O Imagination é um editor de slide show de DVD leve e simples para Linux e FreeBSD. Foi escrito em linguagem C e construído com o GTK + 2 toolkit.

Este aplicativo foi projetado, desde o início, para ser rápido, leve e fácil de usar. Ele exige a instalação do ffmpeg encoder para produzir o arquivo de filme e Libsox para lidar com o áudio. Sim, você não precisa de qualquer outro software de terceiros.

Para efetuar a instalação, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Digite este comando para instalar o imagination
dnf -y install imagination

Para maiores informações e mais detalhes de funcionamento deste aplicativo, acesse o site oficial em Imagination oficial Site

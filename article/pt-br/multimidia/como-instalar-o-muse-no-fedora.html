Como instalar o MuSE no Fedora

MusE é um MIDI/Áudio seqüenciador com gravação e edição escrito originalmente por Werner Schweer e atualmente desenvolvido e mantido pela equipe de desenvolvimento MusE. Funciona como uma estação de trabalho de áudio rico em características digitais.

Tradicionalmente tais dispositivos são referidos como "seqüenciadores", tal como eles combinam sinais de áudio provenientes de várias fontes numa seqüência corretamente cronometrada de saída de som.

O software inclui funções para a gravação e edição de áudio. Tem um misturador, representado em uma interface de utilizador gráfico integrado permitindo a mistura de sinais de áudio e MIDI. Com uma interface flexível, permite visualizar várias partes de uma música simultaneamente.

Para que gosta de um bom editor de musicas esta é uma ótima opção, para efetuar a instalação, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Digite este comando para instalar o MusE
dnf -y install muse

É difícil encontrar bons materiais sobre este aplicativo na internet, porém existe o manual oficial do MuSE no seguinte endereço: Manual MusE e você pode acessar o site oficial através deste link: Site Oficial MusE

Como instalar o cinelerra no Fedora

O Cinelerra é um programa profissional de edição de vídeo originariamente desenhado para os sistemas operativos Linux. É um software livre, sob a licença GPL (General Plublic License) pelo que poderá ser baixado e instalado sem nenhum encargo.

Ele suporta OpenGL em algumas placas gráficas NVIDIA, o que acelera enormemente a reprodução. Atualmente o padrão para a programação GPU mudou ao longo dos anos entre NC, GLSL, e CUDA. A Intel e a ATI/AMD criaram seus próprios padrões. O Cinelarra foi escrito em GLSL, que só está disponível nas placas NVidia e atualmente parece ser a minoria existente no mercado.

Para efetuar a instalação, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Baixe os arquivos do Cinelerra  32 bits
wget ftp://195.220.108.108/linux/atrpms/el6-i386/atrpms/stable/cinelerra-2.1-24.2_git20110613.el6.i686.rpm

# Ou baixe os arquivos do Cinelerra 64 bits
wget ftp://195.220.108.108/linux/atrpms/sl6-x86_64/atrpms/stable/cinelerra-2.1-24.2_git20110613.el6.x86_64.rpm

# Instale o Cinelerra 32 bits no Fedora
dnf -y install cinelerra-2.1-24.2_git20110613.el6.i686.rpm

# Instale o Cinelerra 64bits no Fedora
dnf -y install cinelerra-2.1-24.2_git20110613.el6.x86_64.rpm

Para mais informações, acesse o site oficial em: Site Oficial Cinelerra

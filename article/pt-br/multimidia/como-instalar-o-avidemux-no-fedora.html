Como instalar o AVIDemux no Fedora

O Avidemux é um editor de vídeo gratuito projetado para corte simples, filtragem e codificação tarefas. Ele suporta vários tipos de arquivos, incluindo AVI, MPEG arquivos compatíveis com DVD, MP4 e ASF, usando uma variedade de Codecs. As tarefas podem ser automatizadas usando projetos, fila de tarefas e recursos de script poderoso.

Este aplicativo está disponível para Linux, BSD, Mac OS X e Microsoft Windows sob a licença GNU GPL. O programa foi escrito a partir do zero, mas o código de outras pessoas e projetos tem sido usado também.

Para efetuar a instalação, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Digite este comando para instalar o AviDemux
dnf -y install avidemux

A instalação é bem simples e rápida, se acaso desejar conhecer mais sobre o aplicativo, acesse o site oficial em Avidemux e para acesso a documentação e tutoriais pode acessar este endereço: Wiki Oficial Avidemux

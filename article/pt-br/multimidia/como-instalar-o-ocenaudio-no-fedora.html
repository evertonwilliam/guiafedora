Como instalar o Ocenaudio no Fedora

Ocenaudio é um editor de áudio multi-plataforma de fácil utilização, rápido e funcional. É o software ideal para pessoas que precisam editar e analisar arquivos de áudio sem complicações. Ele também tem as características poderosas que vão agradar os usuários mais avançados.

O Ocenaudio está disponível para todos os principais sistemas operacionais: Microsoft Windows, Mac OS X e Linux. As aplicações nativas são geradas para cada plataforma a partir de uma fonte comum, a fim de alcançar um desempenho excelente e integração com o sistema operacional. Todas as versões do ocenaudio tem um conjunto uniforme de funcionalidades e da mesma interface gráfica, de modo que as habilidades que você aprende em uma plataforma pode ser usada nos outros

Para acelerar a edição de arquivos de áudio complexos, Ocenaudio inclui multi-seleção. Com esta ferramenta incrível, você pode selecionar simultaneamente diferentes partes de um arquivo de áudio e ouvir, editar ou até mesmo aplicar um efeito para eles. 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# baixe o Ocenaudio para plataforma 32 bits
wget http://www.ocenaudio.com/downloads/ocenaudio_centos6_32.rpm

# baixe o Ocenaudio para plataforma 64 bits
wget http://www.ocenaudio.com/downloads/ocenaudio_centos7.rpm

# Instalando a versão 32bits
dnf -y install ocenaudio_centos6_32.rpm

# Instalando a versão 64bits
dnf -y install ocenaudio_centos7.rpm

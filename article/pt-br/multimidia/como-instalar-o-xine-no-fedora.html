como instalar o xine no fedora

O Xine é um reprodutor de mídia de código aberto, disponível para Linux, Solaris e OS/2. Ele executa vários formatos de arquivos multimídia, incluindo os formatos proprietários.

Ele é um software livre com a finalidade de reproduzir arquivos multimídia de diversos formatos, tem o poder de reproduz CDs, DVDs e VCDs como também decodificar arquivos multimídia, como AVI, MOV, WMV e MP3 a partir de disco locais, como também exibir vídeos e músicas via streeming

Para instalar este aplicativo acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Efetue a instalação através do dnf
dnf -y install xine xine-lib

Maiores informações e versões atualizadas do projeto, pode ser visto no site oficial neste link: Xine. Quem tiver interesse de instalar através de arquivos RPM, basta acessar este link: Xine na PUC Rio

como instalar o realplayer no fedora

O RealPlayer é um tocador de mídia de código fechado desenvolvido pela RealTimes destinado à execução de vídeos, músicas e programas de rádio via Internet. Ele reproduz uma série de formatos multimídia, incluindo MP3, MPEG-4, QuickTime, Windows Media e várias versões de codecs RealAudio e RealVideo.

Ele consiste em aplicativos interativos para Internet que realizam diversas comunicações pela Internet como parte de sua operação normal. Vários recursos de comunicação são automáticos e estão ativados por padrão. Por exemplo, quando o RealPlayer, a Atualização Automática e a Central de Mensagens comunicam-se com os servidores da RealTimes, o software transmite informações padrão para que seja recebido o software, o conteúdo, as mensagens e as atualizações apropriadas. Isso inclui informações sobre o produto da RealTimes que tem instalado (incluindo o nome, a versão e o número de build dos produtos), o sistema operacional do usuário, informações sobre sua localidade (incluindo código postal, país e sua preferência de idioma) e informações padrão de comunicação da Internet, incluindo endereço IP, navegador e data.

A primeira versão do RealPlayer foi introduzida em abril de 1995 como RealAudio Player, um dos primeiros executores de mídia capaz de fazer transmissões por streaming sobre a Internet. A versão 4.01 do RealPlayer foi incluída como uma ferramenta da Internet selecionável na instalação do Windows 98. A versão 6 do RealPlayer foi chamada de RealPlayer G2, a versão 9 foi chamada de RealOne Player. O RealPlayer 11 foi lançado para Windows em novembro de 2007, e para Mac OS X em maio 2008.

As versões do RealPlayer também estão disponíveis para Linux, Unix, Palm OS, Windows Mobile e Symbian OS. O programa é alimentado por uma base de mecanismo de mídia de código aberto chamado Helix.

Para você que não fica sem este player, vamos instala-lo acessando o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Efetue a instalação dos componentes 
dnf -y install install redhat-lsb redhat-lsb-core redhat-lsb-desktop

# Agora efetue o Download do Pacote de Instalação
wget http://client-software.real.com/free/unix/RealPlayer11GOLD.rpm

# Agora vamos efetuar a instação
rpm -ivh RealPlayer11GOLD.rpm

# Para abrir o aplicativo, digite este comando
realplay

Após a instalação do RealPlayer, é necessário efetuar a instalação do Pangox, uma extensão do qual o realplayer utiliza para ser executado. Atualmente existem versões desta extensão para algumas versões do Fedora, mas para que não fiquemos amarrados em uma única versão, vamos acessar o FTP do qual existe este aplicativo e baixar a versão exata de nossos Fedora.

Acesse o seguinte endereço de FTP: ftp://195.220.108.108/linux/fedora/linux/releases/, no navegador de sua preferência. Procure pela versão de seu fedora clique no link que corresponde a sua versão, e em seguida vai navegando nas pastas conforme mostrado abaixo:

¤ Clique na pasta: Everything
¤ Se o Fedora for 32 bits clique nesta pasta: i386
¤ Se o Fedora for 64 bits clique nesta pasta: x86_64
¤ Em seguida: os
¤ Depois: Packages
¤ E depois: p
¤ Agora na lista procure pelo aplicativo: "pangox-compat-X.X.X-X.fcXX.XXXX.rpm" onde os X correspondem a versão do aplicativo e a versão do Fedora.
¤ Clique no aplicativo e baixe-o na pasta Downloads de seu usuário e em seguida execute os comandos no terminal do Fedora. 
# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Acesse a pasta Downloads
cd /home/seu-usuario/downloads/

# Instale o aplicativo
dnf -y install pangox-compat-0.0.2-6.fc23.i686.rpm

Prontinho, aconselho reiniciar o fedora, para ativas todas as dependencias instaladas.

Se acaso você ainda não conhece este player, então acesse o site oficial neste link Site Oficial RealPlayer e conheça mais sobre ele.

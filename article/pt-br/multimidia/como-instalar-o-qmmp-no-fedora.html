como instalar o qmmp no fedora

O QMMP é um reprodutor de audio livre e de código aberto escrito em C++ com a ajuda da biblioteca Qt.

Ele suporta oficialmente os sistemas operacionais Linux, FreeBSD e Microsoft Windows. A sua interface padrão é totalmente inspirada no player muito conhecido pelos usuários do Windows, o Winamp.

Para ter este Player leve e muito simples de se usar, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Efetue a instalação através do dnf
dnf -y install qmmp

No site oficial do QMMP é possível encontrar plugins para implementar novas experiências com o aplicativo, para isso acesse o link Site oficial QMMP

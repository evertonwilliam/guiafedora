Como instalar o Kdenlive no Fedora

O Kdenlive é um editor de vídeo gratuito de código aberto para o GNU/Linux e FreeBSD, que suporta DV, AVCHD e HDV. Este software foi projetado para atender a todas as necessidades, desde a edição de vídeo básica até ao trabalho semi-profissional.

Ele contém vários recursos tais como "Edição Multipista", "Vídeo e áudio com faixas organizadas em camadas", "Ferramentas para criar, mover, recortar e excluir clipes de vídeo, clipes de áudio, texto e clips de imagem" entre outras funcionalidades para a edição de vídeos de forma profissional e de alta qualidade.

Para efetuar a instalação, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Digite este comando para instalar o Kdenlive
dnf -y install kdenlive

O manual completo sobre o aplicativo está em seu site oficial, para ter acesso siga o seguinte link: Manual Kdenlive

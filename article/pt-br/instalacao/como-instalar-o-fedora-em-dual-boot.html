como instalar o fedora (dual boot)

Se você tem instalado o Windows em seu computador e não tem interesse em perder esta instalação, então estes passos são para você, para podermos manter uma instalação do Windows intacta e ainda instalar uma versão do Fedora no mesmo computador, precisamos dos seguintes itens:

¤ Um arquivo de imagem ISO do Fedora;
¤ Um pen drive com no mínimo 4gb;
¤ Um aplicativo para configurar o pen drive com o boot do Fedora (imageusb);
¤ Conhecimento para redimencionar o HD se necessário.

Neste caso eu tenho a certeza que você não têm uma cópia do Linux em seu computador, para podemos criar um boot através de um pen drive utilizaremos o Windows, para isso acesse o site: Osforensics.com e baixe o programa ImageUSB, após o download efetue a descompactação do arquivo no diretório "c:\imageusb" ficando assim:

Neste momento vamos escolher a versão do Fedora para baixarmos o arquivo ISO, para isso acesse o seguinte endereço: getfedora.com nessa página podemos ver que existem três versões do Fedora, a versão Workstation, a Server e a Cloud. A versão Workstation é a versão que deveremos baixar, porém esta versão vem com o Gnome, nessa instalação quero utilizar outra versão para ser diferente dos outros manuais, para isso, vamos buscar pelas versões alternativas designadas de SPIN, para acessar estas versões, basta ir até o rodapé da página e clicar na opção "Fedora Spins" como mostra na imagem abaixo:

Na página que apareceu tem as versões SPIN do Fedora, elas são: KDE Plasma Desktop, XFCE Desktop, LXDE Desktop, Mate-Compiz Desktop, Cinnamon Desktop e SOAS Desktop, em particular eu gosto do ambiente Plasma, porém para essa instalação, vou utilizar o LXDE como distribuição padrão, mas se desejar baixar outra SPIN ou até mesmo a versão Workstation não tem problemas, a rotina que vamos seguir nessa instalação independe do ambiente gráfico, então fique á vontade para escolher qual desejar. Para baixar basta clicar na imagem referente à versão desejada e na página que aparecer clique no link DOWNLOAD NOW e depois é só ir seguindo as orientações da tela até chegar na tela que realmente efetua o download do arquivo ISO, como mostra abaixo.

Observe que existem as opções de 64 bits e de 32 bits, escolha a versão correta que irá funcionar em seu computador, se acaso não tiver certeza, baixe a versão 64 bits, se acaso não funcionar, ai tem que ser usada à versão 32 bits. Clique no link que representa o tipo de computador (32 ou 64 bits) e aguarde o download ser totalmente efetuado.

Após o download de seu arquivo ISO o próximo passo é configurar o pen drive para conseguirmos efetuar o boot através dele. Para isso formate o seu pen drive para apagar todos os dados nele contidos e em seguida execute o programa imageusb que você baixou e descompactou no diretório c:\imageusb, será apresentado à seguinte tela:

Na parte superior da tela "Step 1" selecione a letra que corresponde ao seu pen drive, depois na parte central da tela "Step 2" selecione a opção "Write imagem to USB drive" e a opção "Post Image Verification" em seguida no item "Step 3" clique no botão Browse e selecione o arquivo ISO do Fedora, e por fim no "STEP 4" clique no botão Write e aguarde o aplicativo efetuar a gravação em seu pen drive, encerre o aplicativo.

Provavelmente a instalação do Windows está ocupando todo o espaço de seu HD, para que possamos instalar o Fedora em Dual Boot devemos redimencionar o seu HD para que possamos ter uma espaço livre e assim efetuar a instalação do Fedora. Se acaso você for instalar uma cópia limpa do Windows, então na instalação reserve uma partição primária para o Linux, se acaso tenha feito, poderá pular os passos á seguir.

A partir do Windows 7, é possível efetuar o redimencionamento do HD através do Gerenciador do Computador, se acaso estiver utilizando uma versão anterior à essa versão, a solução é procurar por aplicativos que efetuem este tipo de tarefa, uma dica é o Partition Magic que efetua esta terefa de forma bem tranquila e confiável. Em nosso caso, vamos utilizar a própria ferramente do Windows, o Gerenciador do computador.

Para redimencionar o HD acesse o Gerenciador do Computador clicando com o botão direito sobre o ícone "Meu Computador", e no menu que aparecer, selecione a opção "Gerenciar".

Na Janela que aparecer, selecionar no menu à esquerda a opção "Gerenciador de Disco", será apresentado os dispositivos de armazenamento que estão instalados em seu computador no centro da tela.

Escolha o dispositivo do qual está instalado o Windows e clicando com o botão direito sobre a sua partição, escolha a opção Diminuir Volume no menu suspenso que aparecer, aguarde o sistema efetuar a consulta sobre o dispositivo a ser redimencionado.

Após a analise, o Windows irá apresentar a tela "Diminuir" do qual deverá ser inserido o tamanho a ser diminuído do disco, ou seja, se você tem um HD de 250Gb e deseja que 100Gb seja reservado para o Linux, então deverá fazer a seguinte conta: (1024 X 100)=102400 e será este valor que deve ser colocado no campo "Digite o espaço a diminuir em MB". No meu caso, como são apenas 100GB de espaço estou diminuindo 30Gb, ou seja (1.024 X 30)=30720. Observe a tela configurada. Após a definição do tamanho, clique em Diminuir

Após o processo, o Windows terá efetuado o redimencionamento do HD, no tamanho que você definiu, conforme é mostrado na tela abaixo.

Com o Pen Drive espetado em uma das portas USB, reinicie o Windows, não se esqueça de configurar o computador para que reinicie utilizando o pen drive como dispositivo principal de boot. Ao ligar o sistema irá apresentar a seguinte tela:

Selecione a opção "Start Fedora-LXDE-Live 24" utilizando as setas de direções do teclado e aguarde o sistema efetuar toda a carga, será apresentado o ambiente gráfico, do qual você escolheu no download da imagem ISO, no meu caso eu escolhi o LXDE, conforme mostra na tela abaixo.

Se acaso você não quiser instalar de imediato, poderá utilizá-lo em modo Live, porém não poderá instalar novos aplicativos nem atualizar o sistema operacional, então a melhor forma de ter uma boa experiência com o Fedora é instalando ele no disco rígido.

Para iniciar, dê um duplo clique no ícone "Install to Hard Drive" que provavelmente está no menu principal, em alguns ambientes gráficos fica na área de trabalho, será apresentada a tela do Anaconda. Nessa tela selecione a opção "Português" no campo à esquerda e no campo à direita escolha "Português Brasil" conforme é mostrado na tela abaixo, e em seguida pressione o botão "Continuar" na parte inferior da tela.

Nesta nova tela que apareceu, clique na opção "Destino da Instalação" conforme mostra na tela abaixo:

Na tela que apareceu, devemos escolher o HD que será instalado o Fedora, clique duas vezes na imagem que represente o HD que deseja instalar o Fedora. Será liberado na parte inferior a opção "Outras opções de armazenamento", selecionem o item "Eu irei configurar o particionamento" e em seguida clique no botão "Pronto" na parte superior da tela.

Será apresentada a tela: "Particionamento Manual", clique no botão [+] na parte inferior desta tela para que seja apresentado a caixa de configuração da partição. Vamos criar as seguintes partições em nosso sistema de armazenamento, do qual segue as devidas explicações:

¤ Partição swap: Esta partição tem a finalidade de disponibilizar uma área do Disco Rígido para que seja utilizada como memória secundária, ou como conhecido no mundo Windows, como memória de paginação. Antigamente era solicitada a criação da partição SWAP com o dobro da memória RAM, porém, nos dias de hoje, como temos muita memória RAM em nossos computadores, podemos criá-la com um tamanho padrão de 500mb até 1gb. Neste nosso caso criaremos com 1gb.
¤ Partição /boot: Esta partição será utilizada para a gravação de todos os arquivos que são utilizados para a inicialização do sistema Linux Fedora, seu tamanho é de no mínimo 500mb, porém para que possamos ter um espaço bacana para compilações e testes de novos kernels, vou colocar 2gb para ela.
¤ Partição /home: Esta partição é utilizada pelo sistema para a gravação dos arquivos pessoais do usuário, separando esta partição, se for necessário à reinstalação do Fedora, basta não formatar esta partição e todos os seus arquivos pessoais ficarão intactos na nova instalação, ela é opcional, porém estaremos criando-a com o tamanho de 50gb, lembrando que meu HD em questão é de 100gb, escolha o tamanho que quiser para esta partição.
¤ Partição /: esta é a partição raiz do sistema Linux. Temos que criar esta partição com no mínimo 20gb, para podermos ter espaço suficiente para instalarmos nossos aplicativos no Fedora, se você tiver um HD bem grande, eu aconselho que seja reservado no mínimo 100gb para esta partição.

Vamos iniciar nossas configurações, na janela "Adicionar um novo ponto de montagem" selecione no campo "Ponto de Montagem" a opção "swap" e no campo "Capacidade Desejada" digite "512" e clique no botão "Adicionar um ponto de montagem".

Clique novamente no botão [+] na parte inferior da tela e na janela "Adicionar um novo ponto de montagem" selecione no campo "Ponto de Montagem" a opção "/boot" e no campo "Capacidade Desejada" digite "1024" e clique no botão "Adicionar um ponto de montagem"

Clique novamente no botão [+] na parte inferior da tela e na janela "Adicionar um novo ponto de montagem" selecione no campo "Ponto de Montagem" a opção "/home" e no campo "Capacidade Desejada" digite "10240" e clique no botão "Adicionar um ponto de montagem"

Clique mais uma vez no botão [+] na parte inferior da tela e na janela "Adicionar um novo ponto de montagem" selecione no campo "Ponto de Montagem" a opção "/" e no campo "Capacidade Desejada" deixe em branco, para ele assumir todo o restante do HD para esta partição e clique no botão "Adicionar um ponto de montagem"

O sistema deverá ficar com as seguintes configurações, após os passos acima citados:

Feito isso, clique no botão "Pronto" na parte superior da tela. Será apresentada a janela Resumo das Mudanças confira se está tudo conforme desejado e clique no botão "Aceitar Mudanças". O sistema irá gravar as configurações e retornará para a tela "Resumo da Instalação".

O instalador irá apresentar a tela inicial novamente, clique na opção "Teclado", lá deverá verificar se o layout de seu teclado está coerente com o telado de seu computador, para isso faça o teste do layout na caixa à esquerda, testando acentos, cedilhas e caracteres especiais, se acaso não estiver correto, basta clicar no sinal de mais (+) à direita da tela e selecionar o layout correto de seu teclado, após a configuração clique no botão "Pronto" no topo esquerdo da tela.

Novamente o instalador irá voltar para a tela inicial, agora clique na opção "Time e Data" e selecione a região onde você mora através do Mapa, ou utilizando as caixas suspensas na parte superior da tela, após a configuração clique no botão "Pronto.

E para finalizar a configuração, clique na opção "Network e Hostname" e nesta tela, digite o nome que deseja que seu computador apresente caso esteja conectado em uma Rede Local, este nome será apresentado também no terminal de comandos. Escolha um nome sem espaços e sem caracteres especiais em seguida clique em "Pronto", conforme é mostrado abaixo.

Com todos os itens configurados, basta agora clicar no botão "Iniciar a Instalação", localizado na parte inferior da tela, neste momento será iniciada a instalação conforme mostrado abaixo:

Neste momento será apresentado uma nova tela com as opções "Senha Root" e "Criação de Usuário", clique na opção "Senha de ROOT".

Na nova tela, defina uma senha para o usuário root (Insira uma senha e memorize ela, pois esta será solicitada quando necessitarmos de acesso root ao sistema), em seguida, clique em "Pronto" conforme é mostrado na tela abaixo:

O Instalador retornará na tela anterior, clique na opção "Criação de Usuário" e defina um usuário administrador para seu sistema Operacional, entrando com o nome completo, o "User Name" (este deve ser tudo em minúsculo), marque a opção "Tornar este usuário administrador" e digite a senha para este usuário, terminando estes passos clique em "Pronto" e aguarde a conclusão da instalação.

Quando o Anaconda estiver efetuado toda a instalação, será apresentado a mensagem que a instalação foi bem sucedida, basta clicar no botão "Sair".

Em seguida, no menu principal do sistema, clique em "Logout" e quando aparecer a tela de opções de desligamento, clique em Reboot para reiniciar o computador. Quando estiver desligando, remova o pen drive do USB para que o sistema inicie diretamente do disco rígido.

Na Inicialização do sistema, será apresentado o gerenciador de boot, do qual será possível selecionar qual sistema operacional deseja utilizar, conforme mostra na tela à seguir.

Prontinho, viu como é simples? Embora tenha sido vários passos, verificamos que foi muito simples a instalação do Fedora. Agora podemos desfrutar deste novo sistema operacional instalado em seu computador.

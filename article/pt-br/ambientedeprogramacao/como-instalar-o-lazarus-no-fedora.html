como instalar o lazarus no fedora

O Lazarus é uma IDE multi-plataforma compatível com o Delphi para o Desenvolvimento Rápido de Aplicações.

Tem variedade de componentes prontos para uso e um formulário de designer gráfico para criar facilmente interfaces gráficas complexas.

Com o Lazarus pode-se criar navegadores de arquivos, visualizadores de imagem, aplicativos de banco de dados, software de edição de gráficos, jogos, software 3D, software de análise médica ou qualquer outro tipo de software.

Esta plataforma objetiva ser compatível com o Delphi e, ao mesmo tempo, suportar diversas arquiteturas e sistemas operacionais.

O Free Pascal é um compilador de Object Pascal que roda em Linux, Windows, OS/2, Mac OS tradicional, Mac OS X, ARM, BSD, BeOS, DOS e mais. Ele foi desenhado para compilar código com a sintaxe do Delphi ou dos dialetos Pascal do Macintosh e gerar executáveis para diferentes plataformas a partir de um mesmo código-fonte.

Para efetuar a instalação deste aplicativo, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Instalando o Lazarus
dnf -y install lazarus

Acesse o site oficial do lazarus do qual é possível efetuar o download da ferramenta para outras versões do linux, como também para Mac-OSX, Windows e o código fonte para compilação em outras plataformas como por exemplo o BSD.

como instalar o aptana no fedora

O Aptana ou Aptana Studio é um software open source gratuito para IDE (Integrated Development Environment), desenvolvido em Java que suporta as linguagens PHP, Python, Ruby, Ruby on Rails, CSS3, HTML5, JavaScript, ScriptDoc, XML e texto comum, embora também seja possível configurá-lo para suportar Adobe® AIR e Bibliotecas AJAX.

É baseado no IDE Eclipse, software que tem a finalidade principal de desenvolver aplicativos para Java e Android, porém com plugins é possível acrescentar uma infinidade de outras linguagem de programação.

Aptana Studio é distribuído em multiplataforma, ou seja, pode ser rodado em Windows, Linux e Mac OS X, como também é possível integrá-lo ao Eclipse através de um plugin específico.

Uma das funcionalidades bastante interessante do Aptana é o assistente de código. Ele funciona como um quadro branco, estilo pop-up, que sugere como você pode preencher com o código. Funciona da seguinte maneira: O desenvolvedor digita uma tag e o Aptana mostra a possibilidade de auto-completar aquela tag e em que navegadores ela está disponível para ser renderizada corretamente. Junto do auto-completar, é exibida também uma caixa de descrição contendo o modo de preencher aquele código.

Antes de efetuar a instalação do Aptana, é necessário efetuar a instalação do Java, se acaso não estiver instalado, para isso acesse este link: "Como instalar o Java no Fedora" e após a instalação do "Oracle Java" ou do "Open JDK" execute os seguintes passos abaixo.

¤ Baixe o Aptana via navegador no através do seguinte link: Download Aptana;
¤ Acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos;
# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Acesse a pasta Downloads (considerando que foi aí que salvou o Download)
cd /home/seu-usuario/Downloads

# Descompacte o arquivo zip baixado
unzip Aptana_Studio_3_Setup_Linux_x86_64_3.6.1.zip

# mova o diretório descompactado do Aptana para o diretório de binários
mv Aptana_Studio_3/ /usr/local/bin

# Conceda as permissões de acesso ao Aptana
find Aptana_Studio_3/ ! -type d -exec chmod a+r {} \;

# Crie o link no menu principal
touch /usr/share/applications/aptana3.desktop

# Edite o arquivo criado, colando as seguintes linhas dentro dele
nano -w /usr/share/applications/aptana3.desktop

[Desktop Entry]
Encoding=UTF-8
Name=Aptana studio 3
Comment=Aptana studio 3
Exec=/usr/local/bin/Aptana_Studio_3/AptanaStudio3
Icon=/usr/local/bin/Aptana_Studio_3/icon.xpm
Terminal=false
Type=Application
Categories=Development;Utility;TextEditor;
StartupNotify=true
Actions=Window;Document;
MimeType=text/plain;text/html;text/x-php;text/x-c;text/x-shellscript;

Se acaso, ao tentar descompactar o arquivo baixado, o Fedora lhe apresentar a mensagem que o comando unzip não existe, então deverá efetuar a instalação do unzip, para isso, acesse o link: "Instalando os Compactadores Rar/Zip" e efetue os passos lá apresentados.

como instalar o apache tomcat no fedora

O Tomcat é um servidor web Java, mais especificamente, um container de servlets. Ele implementa, dentre outras de menor relevância, as tecnologias Java Servlet e JavaServer Pages (JSP) que são tecnologias criadas para o desenvolvimento de aplicações com a Linguagem JAVA.

Criado pela Apache Software Foundation, é distribuído como software livre. Hoje um projeto independente, porém, foi criado dentro do projeto Apache Jakarta e posteriormente separado, devido a conclusão do projeto Jakarta.

Ele cobre parte da especificação Java EE com tecnologias como servlet e JSP, e tecnologias de apoio relacionadas como Realms e segurança, JNDI Resources e JDBC DataSources. Ele tem a capacidade de atuar também como servidor web, ou pode funcionar integrado a um servidor web dedicado como o Apache ou o IIS. Como servidor web, ele provê um servidor web HTTP puramente em Java.

Antes de efetuar a instalação do Tomcat, é necessário efetuar a instalação do Java, se acaso não estiver instalado, para isso acesse este link: "Como instalar o Java no Fedora" e após a instalação do "Oracle Java" ou do "Open JDK" execute os seguintes comandos abaixo. 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Instalando o Tomcat
dnf -y groupinstall tomcat

# Habilitando para iniciar com o Fedora
systemctl enable tomcat

# Iniciando o serviço do tomcat
service tomcat start

# Parando o serviço do tomcat
service tomcat stop

# Reiniciando o serviço do Tomcat
service tomcat restart

Se acaso o Java não estiver instalado e for efetuado a instalação do Tomcat, o DNF irá efetuar automaticamente a instalação do OpenJDK, pois faz parte das dependências do Tomcat. Para saber mais sobre o projeto, acesso o Site Oficial.

como instalar o Apache OpenOffice no Fedora

OpenOffice.org é um conjunto de aplicativos para escritório livres multiplataforma, distribuída para Microsoft Windows, Unix, Solaris, Linux e Mac OS X, mantida pela Apache Software Foundation. O conjunto usa o formato ODF (OpenDocument) - formato homologado com o ISO/IEC 26300 e NBR ISO/IEC 26300 e é também compatível com os formatos do Microsoft Office, além de outros formatos legados. Alguns formatos legados não mais suportados pelas versões mais recentes do Microsoft Office ainda podem ser abertos pelo OpenOffice.org evitando assim sua perda.

O OpenOffice.org é baseado em uma antiga versão do StarOffice, o StarOffice 5.1, adquirido pela Sun Microsystems em Agosto de 1999. O código fonte da suíte foi liberado dando início a um projeto de desenvolvimento de um software de código aberto em 13 de outubro de 2000, o OpenOffice.org. O principal objetivo era fornecer uma alternativa de baixo custo, de alta qualidade e de código aberto.

O OpenOffice.org é composto pelos seguintes aplicativos: OpenOffice.org Writer, OpenOffice.org Impress, OpenOffice.org Draw, OpenOffice.org Base.

Para efetuar a instalação deste aplicativo, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Baixe o OpenOffice pt-br versão 64bits
wget http://sourceforge.net/projects/openofficeorg.mirror/files/4.1.1/binaries/pt-BR/Apache_OpenOffice_4.1.1_Linux_x86-64_install-rpm_pt-BR.tar.gz/download -O Apache_OpenOffice_4.1.1_Linux_x86-64_install-rpm_pt-BR.tar.gz

# Para fins de compatibilidade remova o OpenOffice e o LibreOffice se acaso estiverem instalados
dnf remove openoffice* libreoffice*

# Extraia os arquivos de instalação
tar -xvf Apache_OpenOffice_4.1.1*

# Acesse a pasta pt-BR
cd pt-BR

# Edite o arquivo yum para futuros updates
nano -w /etc/yum.conf
 
# Adicione a seguinte linha
[main]
exclude=openoffice.org-ure* libreoffice-ure*

# Digite este comando para instalar o OpenOffice
rpm -Uvh RPMS/*.rpm RPMS/desktop-integration/openoffice4.1.1-redhat-*.rpm

# Acesse o Aplicativo via terminal
openoffice4

Viu, embora pareça complicado, não foi preciso ficar acessando sites de Keygeration para ativar o aplicativo, e ainda por cima, não precisa ficar se preocupando com pirataria, uma vez que ele é totalmente gratuíto.

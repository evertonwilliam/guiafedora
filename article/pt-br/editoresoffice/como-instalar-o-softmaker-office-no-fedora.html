como instalar o softmaker office no fedora

O SoftMaker Office é um conjunto de aplicativos de escritório desenvolvido desde 1987 pela SoftMaker Software GmbH, com sede em Nuremberga, na Alemanha.

A Versão gratuita do SoftMaker Office é licenciada para utilização particular e profissional, para Windows e Linux, e distribuído com um processador de texto, uma planilha e um programa de apresentações gráficas. Possui maior compatibilidade com o Microsoft Office do que o LibreOffice.

Esta suite têm muitas funções iguais ou similares aos conhecidos MS Office, OpenOffice.org ou WordPerfect Office, e também pode ser executado a partir de drives flash USB. Ele tem seus próprios formatos nativos, e pode ler e escrever formatos de arquivo do Microsoft Office, OpenDocument Format (processador de texto somente), RTF e HTML. Ele pode exportar para PDF.

Para efetuar a instalação deste aplicativo, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# Baixe o arquivo para a versão 32 bits - Se o Fedora for 32 bits
wget http://www.softmaker.net/down/softmaker-office-2016-749.i386.rpm

# Baixe o arquivo para a versão 64 bits - Se o Fedora for 64 bits.
wget http://www.softmaker.net/down/softmaker-office-2016-749.x86_64.rpm

# Instale a versão 32 bits - Se o Fedora for 32 bits
rpm -ivh softmaker-office-2016-749.i386.rpm

# Instale a versão 64 bits - Se o Fedora for 64 bits
rpm -ivh softmaker-office-2016-749.x86_64.rpm

O SoftMaker Office é um software pago, desta forma a versão que estamos baixando é trial e será expirada após um determinado tempo, sendo assim, se gostar do aplicativo, deverá acessar o site oficial e efetuar a compra do mesmo.

Como Instalar o Mozilla Firefox no Fedora

Firefox é um dos navegadores mais famosos do mercado ele é desenvolvido para navegar na Internet utilizando os seguintes sistemas Operacionais: OS X, Linux e Android, ele é mantido pela Fundação Mozilla.

Mozilla Firefox é um navegador livre e multi-plataforma desenvolvido pela Mozilla Foundation (em português: Fundação Mozilla) com ajuda de centenas de colaboradores. A intenção da fundação é desenvolver um navegador leve, seguro, intuitivo e altamente extensível. Baseado no componente de navegação da Mozilla Suite (continuada pela comunidade como Seamonkey), o Firefox tornou-se o objetivo principal da Mozilla Foundation.Cerca de 40% do código do programa foi totalmente escrito por voluntários.

Vamos então instalá-lo, o Firefox está nos repositórios oficiais do Fedora, alguns dos SPINS, se não a maioria, já vem com ele instalado, se acaso você escolheu o SPIN KDE-Plasma este navegador não vem instalado, pois nele vem por padrão o konqueror, antes de executar os passos abaixo verifique se ele está ou não está instalado. Para instalá-lo basta invocar o comando dnf e o nome do aplicativo. Para isso vamos acessar o terminal Konsole pelo seguinte menu: Aplicativos->Sistema->Terminal e digitar os seguintes comandos. 

# Acesse o modo superusuario.
su

# Digite a sua senha de superusuário
Senha

# Instalando o Firefox
dnf -y install firefox

Após a instalação, é possível acessar o navegador no seguinte caminho: Aplicativos->Internet->Firefox

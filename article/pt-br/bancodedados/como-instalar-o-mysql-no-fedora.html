como instalar o SGDB Mysql no Fedora

O SGDB MySQL não convive juntamente com o MariaDB. Para que seja instalado o MySQL no Fedora é necessário efetuar a desinstalação do MariaDB. Se acaso você desejar manter o MariaDB instalado, então não execute os passos listados neste manual pois o MariaDB será desinstalado e totalmente substituído pelo MySQL.

Para efetuar a instalação do MySQL no Fedora, devemos primeiramente desabilitar o repositório do MariaDB, em seguida efetuar a desinstalação total do MariaDB. Após a desinstalação do MariaDB deve-se efetuar a instalação do repositório do MySQL, este arquivo de instalação é encontrado no site do MySQL pelo endereço: http://dev.mysql.com/downloads/repo/yum/.

Abaixo segue os passos para desinstalar o MariaDB, para executar abra o terminal de comandos do Fedora através do menu: "Aplicativos->System Tolls->LXTerminal" e digite os seguintes comandos: 

# acesse a conta root
su

#digite a sua senha:
Senha:

# Desabilitando as entradas dnf para o MariaDB
dnf-config-manager --disable mariadb

# Efetuando a remoção do MariaDB
dnf remove -y mariadb-common mariadb-compat mariadb-server

Antes de iniciarmos a instalação do MySQL, devemos efetuar o download e a instalação de seu repositório, para isso efetue o download do arquivo rpm de instalação dos repositórios da Oracle, para isso acesse o site: Oracle Mysql será solicitado um login para efetuar o download, se já tiver, basta efetuar o login e o arquivo será baixado, senão poderá efetuar o seu cadastro sem nenhum custo.

Após o Download do arquivo RPM, siga os passos á seguir, no terminal do Fedora, estando logado como ROOT. 

# Efetue a instalação do repositório do MySQL (ver explicações abaixo)
dnf install -y mysql-community-release-fc21-6.noarch.rpm

# Efetue a verificação do repositório
dnf repolist enabled | grep "mysql.*-community.*"

# Efetue a instalação do MySQL no Fedora
dnf install -y mysql-community-server

# Depois da instalação, efetue a configuração da senha
mysql_secure_installation

Escolha o RPM que representa a versão de seu Fedora, lá tem os arquivos designados para as versões atuais do fedora.

Feito isso, os passos de acesso ao MySQL são os mesmos apresentados no MariaDB.

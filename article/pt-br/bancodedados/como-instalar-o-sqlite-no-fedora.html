como instalar o SQLite no Fedora

O SQLite é uma biblioteca em linguagem C que implementa um banco de dados SQL embutido. Programas que usam a biblioteca SQLite podem ter acesso a banco de dados SQL sem executar um processo SGBD separado.

Ele não é uma biblioteca cliente usada para conectar com um grande servidor de banco de dados, mas sim o próprio servidor. A biblioteca SQLite lê e escreve diretamente para e do arquivo do banco de dados no disco.

O uso do SQLite é recomendado onde a simplicidade da administração, implementação e manutenção são mais importantes que incontáveis recursos que SGBDs mais voltados para aplicações complexas possivelmente implementam. As situações onde a simplicidade é a melhor escolha são muito mais freqüentes do que pode-se imaginar

A instalação do SQLite é bem simples, ele é um banco de dados muito leve e muito eficiente, por ele ser executado um aplicativo não é necessário ser iniciado e nem parado como acontece com os outros SGDBs.

Para efetuar a instalação deste aplicativo, acesse o terminal através do menu: "Aplicativos->System Tolls->LXTerminal" e na tela que aparecer digite os seguintes comandos: 

# acesse a conta root
su

#digite a sua senha:
Senha:

# Instalando o SQLite
dnf install -y sqlite sqlite-devel

# Acessando o SQLite
sqlite3

#Irá aparecer a seguinte mensagem no Prompt
SQLite version 3.8.8.3 2015-02-25 13:29:11
Enter ".help" for usage hints.
Connected to a transient in-memory database.
Use ".open FILENAME" to reopen on a persistent database

# Para sair do modo interativo no SQLite digite
.quit
